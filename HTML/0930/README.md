## ⭐코드⭐
    [ Bulma 사용 ]
    
    HTML에서 사용하기 위해 head에 아래 코드 추가
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.4/css/bulma.min.css">
    
    
    [ jQuery Mobile 테마 ]
    태그에 data-theme 속성을 지정하면 테마 변경 가능
    
    data-theme="a"  : jQuery Mobile의 기본 테마
    data-theme="b"
